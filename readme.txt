LoadRunner (TruClient) script witch passed http://test.youplace.net/ test.
How to use:
	1. Download and extract file TruClient1.zip.
	2. In TruClient1 folder launch TruClient1.usr file with Virtual User Generator 12.60 build 0582 
	3. Press "Develop Script" (Ctrl+R) from toolsbar.
	4. Start "Replay" (Ctrl + Alt + F5) and Enjoy.
